# Signature Generator

## Introduction

This repo contains a simple signature generator for email signatures. It is written in `HTML` and uses the `css`.

## Usage
```
folder_name/
│
├── README.md
│
├── html_signature/
│   ├── index.html
│   └── styles.css
│
└── assets/
    ├── images/
    │   └── logo.png
    └── fonts/
        └── font.ttf
```
```
mkdir folder_name
cd folder_name
touch README.md
mkdir html_signature
cd html_signature
touch index.html
touch styles.css
cd ..
mkdir assets
```
